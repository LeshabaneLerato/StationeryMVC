# StationeryMVC

A web-based Stationery Management and Quotation application built using **ASP.NET Core MVC**.  
The system allows users to manage stationery items, generate QR codes, create quotations for customers, and manage shop settings.

## Features
- Add, edit, view, and delete stationery items
- Store item details including name, category, quantity, price, image, and QR code
- Generate and store QR codes for items
- Create quotations for customers
- Calculate total amount for quotations
- Store quotation date and customer information
- Manage shop settings: shop name, slogan, logo
- Dashboard to display summary of items and quotations

---

## System Requirements
- .NET 6.0 or higher
- Visual Studio 2022 or VS Code
- SQL Server Express or LocalDB
- Modern browser (Chrome, Edge, Firefox)
